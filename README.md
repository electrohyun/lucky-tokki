# 🍀 LuckyTokki (러키토끼) 🐰

간단한 iOS 앱으로, 사용자가 생일을 선택하면 해당 별자리와 오늘의 운세를 보여줍니다. 한국어 UI와 메시지를 제공하며, 간단한 버튼과 날짜 선택기를 통해 결과를 확인할 수 있습니다.

## ✨ 주요 기능
- 날짜 선택(UIDatePicker)으로 생일 설정
- 생일에 따른 12별자리 판별
- 별자리 이모지와 함께 랜덤 운세 메시지 표시
- 커스텀 폰트(LeeSeoyun) 적용

## 📺 화면 구성
- 상단 안내 레이블: 앱 사용 방법 안내
- 생일 레이블 및 버튼: 현재 선택된 생일 표시, 버튼으로 날짜 선택기 표시/숨김 토글
- 날짜 선택기(휠 스타일): 생일 선택
- 결과 레이블: 별자리와 운세 메시지 표시 (버튼 탭 후 노출)
- 토끼 버튼: 운세 보기 트리거 버튼 (이 예제에서는 운세 표시 버튼)

## ⚙️ 동작 방식
1. 앱 실행 시 날짜 선택기는 숨겨져 있으며, 기본 날짜는 오늘로 설정됩니다.
2. "생일" 버튼을 탭하면 날짜 선택기가 나타나고, 사용자가 날짜를 바꾸면 레이블이 업데이트됩니다.
3. "토끼" 버튼을 탭하면:
   - 현재 선택된 생일을 기준으로 별자리를 계산합니다.
   - 별자리에 맞는 이모지와 랜덤 운세 메시지를 가져와 결과 레이블에 표시합니다.
   - 생일 관련 UI(레이블/버튼/데이트피커)는 숨기고 결과 레이블만 보여줍니다.

## 💻 코드 개요
앱의 핵심 로직은 `HoroscopeViewController.swift`에 구현되어 있습니다.

- `selectedBirthday: Date`: 현재 선택된 생일을 보관합니다.
- `horoscopes: [String: [String]]`: 별자리별 운세 메시지 사전입니다(한국어).
- `viewDidLoad()`: 날짜 선택기 기본 설정, 초기 레이블 업데이트, 커스텀 폰트 적용.
- `applyCustomFonts()`: `LeeSeoyun` 폰트를 주요 UI 요소에 적용합니다.
- `updateBirthLabel()`: 현재 선택된 날짜를 한국어 형식("yyyy년 M월 d일")으로 표시합니다.
- `didTapBirthButton(_)`: 날짜 선택기의 표시/숨김을 토글합니다.
- `datePickerChanged(_)`: 날짜가 변경될 때 `selectedBirthday`를 갱신하고 레이블을 업데이트합니다.
- `didTapRabbitButton(_)`: 별자리 계산 → 이모지 선택 → 랜덤 운세 메시지 표시 → 결과 화면 구성.
- `zodiacSign(for:)`: 입력된 날짜의 월/일을 기준으로 12별자리(양자리~물고기자리)를 반환합니다.
- `zodiacEmoji(for:)`: 별자리명에 대응하는 이모지를 반환합니다.

## ⭐️ 별자리 기준
아래 범위를 기준으로 별자리를 판별합니다.
- 양자리: 3/21–4/19
- 황소자리: 4/20–5/20
- 쌍둥이자리: 5/21–6/21

- 천칭자리: 9/24–10/22
- 전갈자리: 10/23–11/22
- 사수자리: 11/23–12/21
- 염소자리: 12/22–1/19
- 물병자리: 1/20–2/18
- 물고기자리: 2/19–3/20

## 📱 요구 사항
- iOS 13 이상 (UIKit 기반)
- Xcode 14 이상 권장
- 커스텀 폰트 `LeeSeoyun`을 사용하려면 앱 번들에 폰트 파일을 포함하고 `Info.plist`의 `Fonts provided by application`에 등록해야 합니다.

## ⬇️ 설치 및 실행
1. 저장소를 클론하거나 프로젝트를 다운로드합니다.
2. Xcode로 `.xcodeproj` 또는 `.xcworkspace`를 엽니다.
3. (선택) 커스텀 폰트 `LeeSeoyun`을 프로젝트에 추가하고 Info.plist에 등록합니다.
4. 시뮬레이터 또는 실제 기기에서 빌드/실행합니다.

## 💁‍♀️ 커스터마이징 팁
- 운세 메시지: `horoscopes` 딕셔너리에 항목을 추가/수정하여 메시지를 늘릴 수 있습니다.
- 이모지/아이콘: `zodiacEmoji(for:)`에서 원하는 이모지로 변경할 수 있습니다.
- 날짜 포맷/로케일: `updateBirthLabel()`의 `DateFormatter` 설정을 수정하여 다른 언어/형식을 지원할 수 있습니다.
- UI/레이아웃: 스토리보드 또는 코드에서 버튼/레이블 스타일을 조정하세요.
